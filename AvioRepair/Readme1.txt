Руководство пользователя

Проект: Назначение моего ПП

Назначение:
Данная программа предназначена для учета информации о пользователях, входа пользователей, цехах, сотрудниках, оборудования, видов работ и ремонтов. 
Позволяет быстро находить нужные данные и управлять ими.

Основные разделы программы:
Администрирование - включает данные о пользователях: логин, пароль, права.
Журнал регистрации - включает данные о входах пользователей систему.
Цехи - список цехов и их адрес.
Сотрудники - список сотрудников, их ФИО, должность оклад и т.д.
Оборудование - список оборудования, их тип и состояние.
Виды работ - включает данные о производимых работах.
Ремонты - общие данные о состоянии ремонта.


Для того чтобы пользователю начать работу с программой, нужно:
1. Открыть файл AvioRepairApp.sln.
2. После открытия файла пройти авторизацию.
3. После авторизации вы попадёте на главную страницу, через которую можно проводить работу, которая доступна пользователю. 
4. После перехода на страницу по нажатию кнопки с соответственным названием «Ремонты».
5. Пользователь может перейти на страницу для добавления/редактирования данных, используя, кнопку «Добавить».
6. После ввода указанных полей пользователь может сохранить данные в таблице нажатием кнопки «Сохранить» или отменить ввод данных нажатием кнопки «Назад».
7. Для изменения данных необходимо нажать кнопку «Изменить» на соответствующей записи.
8. Дальнейшие действия аналогичны добавлению.
9. Удаление данных происходит после нажатия кнопки «Удалить» у тех данных которые необходимо удалить и дальнейшего подтверждения пользователя во всплывающем окне.
10. Поиск производится в реальном времени при изменении текстового поля, предназначенного для поиска, в случае отсутствия записей выдаётся сообщение, для сброса поиска необходимо очистить данное поле.
11. Фильтрация производится в реальном времени при изменении соответствующего выпадающего списка и выбора необходимого параметра. Отмена производится выбором среди выпадающего списка записи.
12. Для получения необходимого отчёта о заявках необходимо после нажатия на странице кнопки «Сформировать отчёт» перейти на страницу формирования отчёта при необходимости выбрать временной промежуток и нажать кнопку «Сформировать отчёт» после чего он сформируется автоматически на этот промежуток времени, в противном случае отчёт сформируется за всё время. 
13. Работа с другими таблицами осуществляется аналогично.

Советы
Используйте поиск для быстрого нахождения нужных данных.
Регулярно проверяйте и обновляйте информацию.
Для добавления новых данных используйте соответствующие формы.
